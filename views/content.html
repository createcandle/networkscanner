The Network Scanner does not fully start until two minutes after a (re)boot. This is to avoid interfering with other addons that may need the network to get set up properly.

Network vulnerability scan

Smart home devices are infamous for their poor security. If they run outdated software they may be susceptible to hacking and other types of abuse. To find out if any devices are vulnerable you can run a scan on networks that the Candle Controller is connected to.

Before you continue:

  • Make sure you have permission from the owner of each device on your network.
  • This scan requires an internet connection, as it reaches out to a server that has a huge list of known vulnerabilities (details below).
  • Be aware that this scan is intensive, and can temporarily cause network slowdown or other disruptions.
  • Stay calm. The scan detects security vulnerablities, but can't tell if devices have actually been compromised. Actual risk of abuse depends on many more factors.
  • Be patient. The scan will take quite a while.
Learn more

How it works:

  1. If it's been a while then the security scan itself will be updated first. In case you're curious, this will download the latest version of the following file:

    https://svn.nmap.org/nmap/scripts/vulners.nse

    The security scan was last updated on: ?
  2. The scan generates an inventory of your network's devices and their open ports, and ascertains which follow-up scans may be useful.
  3. It then reaches out to a webserver that has a huge list of known vulnerabilities (their database is over 250 gigabytes!), and how to try to detect them. Learn more at:

    https://nmap.org/nsedoc/scripts/vulners.html.
  4. It downloads specific tests for the most likely vulnerabilities, and runs them.
  5. When all is done you are shown a report.


Debugging is enabled

This addon is currently set to save detailed information about its operation in the internal logs. This may impact your privacy, as this also means that insights into your behaviour might be stored in those logs.

You can disable debugging mode at the settings page for this addon.
